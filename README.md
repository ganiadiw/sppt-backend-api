## Penggunaan

```
composer install
```
```
Windows (CMD) : copy .env.example .env | Linux (Bash) : cp .env.example .env
```
```
php artisan key:generate
```
Configure your database in .env file
```
php artisan migrate
```
```
php artisan db:seed
```
```
php artisan storage:link
```
```
php artisan serve
```
